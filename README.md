## Quantiative data
- Data used to quantify a problem
- Can be counted or compared on numeric scale
- usually gathered using instruments 
  1. Rating scale
     - questionnaire
  2. Thermometre
     - Harvest Weather data
  3. Statistical software 
     - SPSS used for quantitative data analysis
